# Micro-Training PWA

A lightweight, offline-first Progressive Web App designed to guide a daily micro-training system for people who spend many hours sitting.

## What this app does
- Manual start (no calendar dependency)
- Lets you choose a logical training day (Lunes, Martes, Jueves, Viernes)
- Guides you through 5 micro-training pauses
- Waits 1 hour between pauses
- Plays an audible alert when the next pause is ready
- Works offline once installed
- Can be installed as a Windows desktop app (Chrome / Edge)

## Technology
- HTML
- CSS
- Vanilla JavaScript
- Service Worker
- LocalStorage
- GitHub Pages hosting

No backend. No frameworks.

## How to use
1. Open the app:
   https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/

2. Install it from Chrome or Edge using the “Install app” button.

3. Start the routine manually and follow the pauses.

## Notes
- Training days are logical, not calendar-based.
- All timing is relative to manual start.
- State is persisted locally in the browser.
